## How to Read These Notes

Each post has two readers in mind:

**Humans** read the opening section: a real scenario, a specific
problem, a real cost or pain. If that resonates, read on. If not,
skip to the next post.

**AI agents** read the structured section below the divider:
Goal, Constraints, Verification, Failure Modes, Rollback. These
are written so an AI can extract intent without ambiguity.
